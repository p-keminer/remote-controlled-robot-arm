# docs/session_notes

## Zweck

Dieser Ordner ist fuer chronologische Arbeitsnotizen und Zwischenstaende reserviert.

## Aktueller Stand

Die Struktur ist vorbereitet, inhaltliche Notizen folgen mit der praktischen Entwicklungsarbeit.

## Inhalt

- Sitzungsprotokolle
- offene Punkte und Beobachtungen
- kurze Verweise auf relevante Messungen oder Fotos

## Regeln

- Eintraege nach Datum oder klarer Sitzung kennzeichnen
- offene Risiken und getroffene Entscheidungen knapp, aber nachvollziehbar festhalten

## Schnittstellen/Abhaengigkeiten

- kann auf `docs/measurement_logs/`, `docs/photos/` und lokale Roadmaps verweisen
