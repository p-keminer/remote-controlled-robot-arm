# official_downloads/raw

## Zweck

Dieser Ordner enthaelt nur den Verweis auf die offiziellen Bezugsquellen des Adeept-Herstellerpakets.
Die rohe ZIP-Datei selbst wird nicht dauerhaft im Repository gehalten.

## Offizielle Herstellerlinks

- Direkter Produkt- und Kaufpfad: [Adeept Robotic Arm UNO](https://www.adeept.com/robotic-arm-uno_p0118.html)
- Offizieller Download- und Lernpfad: [Adeept Learn Detail 64](https://www.adeept.com/learn/detail-64.html)

## Regeln

- grosse Roharchive wie `.zip` werden hier nicht eingecheckt
- wenn ein Herstellerdownload lokal ausgewertet wird, bleibt der Nachweis ueber die offiziellen Links und die projektspezifische Auswertung in `../ADEEPT_V4_PACKAGE_REVIEW.md`
- relevante Erkenntnisse aus neuen Herstellerstaenden muessen in die Fach- und Managementdokumente nachgezogen werden
