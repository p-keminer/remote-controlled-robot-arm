# official_downloads

## Zweck

Dieser Ordner archiviert unveraenderte Hersteller- und Vendor-Staende als Referenzbasis.

## Aktueller Stand

Der Bereich bleibt bewusst vom aktiven Projektstand getrennt.
Er beschreibt also nicht den heutigen Prototyp, sondern den Hersteller-Ursprung, aus dem das Projekt hervorgegangen ist.

Das ist wichtig, weil der aktive Projektstand inzwischen u.a. zusaetzliche ESP32-Knoten, Potentiometer als Greifer-Eingabe, Perfboard und digitale Zwillinge umfasst.

## Regeln

- Vendor-Inhalte bleiben unveraendert
- aktive Projektentscheidungen werden in den Hauptdokumenten gepflegt
- `raw/` und `extracted/` werden nicht in den generierten Snapshot gespiegelt
