# tests/bench

## Zweck

Dieser Ordner sammelt kleine, fokussierte Teilpruefungen.

## Aktueller Stand

Historische Bench-Tests fuer IMUs, Mux und Funk bleiben relevant.
Der aktuelle Prototypstand fuegt dazu die Greifer-Eingabe per Potentiometer und den Perfboard-Controller als aktive Testbasis hinzu.

## Regeln

- Bench-Tests bleiben klein und eindeutig.
- Historische Breadboard-Ergebnisse werden von aktuellen Prototyp-Checks getrennt beschrieben.
