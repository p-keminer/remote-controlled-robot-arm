# Roadmap - hardware

## Ziel

Eine nachvollziehbare Hardwaredokumentation fuer Elektrik, Verdrahtung, Aufbauablauf, Bringup und Materialstand aufbauen.

## Arbeitspakete

- [x] Hardwarebereiche und Unterordner festlegen
- [x] Aufbau- und Bringup-Bereiche als Dokumentationsstruktur einfuehren
- [ ] Schaltplanstand fuer Controller- und Armseite dokumentieren
- [ ] reale Verkabelung gegen den Schaltplanstand abgleichen
- [ ] mechanische Halterungen und Befestigungen beschreiben
- [ ] Stueckliste fuer Prototyp und spaeteren Daueraufbau pflegen
- [ ] Inventar-, Aufbau- und Einschaltvorlagen spaeter mit realen Nachweisen fuellen
