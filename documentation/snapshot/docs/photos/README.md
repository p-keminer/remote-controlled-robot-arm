# docs/photos

## Zweck

Dieser Ordner enthaelt Bildnachweise fuer Aufbau, Montage, Sensorlage und Verkabelung.

## Aktueller Stand

Die Struktur ist vorbereitet, Bildmaterial wird spaeter hier abgelegt.

## Inhalt

- Fotos des Wearable-Aufbaus
- Fotos der IMU-Ausrichtung
- Fotos der Verkabelung und Armseite
- Bildnachweise fuer Referenzposen

## Regeln

- Bilder so benennen, dass Datum, Zweck oder Aufbauzustand erkennbar sind
- wenn ein Foto eine Kalibrier- oder Sicherheitsentscheidung stuetzt, sollte es querreferenziert werden

## Schnittstellen/Abhaengigkeiten

- ergaenzt `hardware/`, `calibration/reference_poses/` und `docs/session_notes/`
