# preparation/esp32_environment

## Zweck

Dieser Ordner beschreibt die vorbereitende Umgebung fuer die beiden ESP32-Komponenten.

## Aktueller Stand

Die Struktur ist vorbereitet.
Reale Setup- und Verifikationsschritte fuer Boardpakete, Bibliotheken und serielle Erreichbarkeit folgen spaeter.

## Inhalt

- benoetigte ESP32-Grundlagen
- erwartete Bibliotheksbasis fuer Sensorik und `ESP-NOW`
- Checkliste fuer erste Entwicklungsbereitschaft

## Regeln

- Toolchainannahmen klar von realen installierten Zustaenden trennen
- keine implizite Abhaengigkeit zu WLAN-Setups dokumentieren

## Schnittstellen/Abhaengigkeiten

- bezieht sich auf `../../firmware/esp32_controller/`, `../../firmware/esp32_receiver/` und `../../security/`
