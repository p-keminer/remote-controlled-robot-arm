# preparation/arduino_ide_setup

## Zweck

Dieser Ordner beschreibt den ersten vorgesehenen Entwicklungsworkflow ueber die Arduino IDE.

## Aktueller Stand

Die Dokumentationsbasis ist angelegt.
Der eigentliche lokale Setup-Stand wird spaeter anhand realer Installationsschritte gepflegt.

## Inhalt

- vorbereitende Schritte fuer Arduino IDE und Board Manager
- Nachweis, welche Boards und Bibliotheken fuer Controller, Receiver und Arduino-Seite benoetigt werden
- einfache “erledigt/nicht erledigt”-Pruefpunkte fuer die Einsatzbereitschaft

## Regeln

- keine Versionsannahmen ohne Nachweis festschreiben
- reale Installationsstaende und Abweichungen spaeter dokumentieren

## Schnittstellen/Abhaengigkeiten

- bezieht sich auf `../../firmware/` und `../../docs/templates/`
